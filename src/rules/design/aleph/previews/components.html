01 · Buttons

Action vocabulary

Primary for the one important action per screen. Secondary for everything else. Ghost for low-emphasis links inside dense rows. Danger only for destructive verbs (halt, delete, archive).

02 · Badges & chips

Named state, named priority

Status is named directly: active, paused, done. Lowercase. No icons replacing words. Priority and state badges share visual weight; categories are slightly lighter.

03 · Inputs & forms

Quiet fields, mono labels

Field labels in mono caps signal "system field." Placeholders carry an example, not just a verb. Multiline inputs lean tall — research prompts can run several lines.

04 · Cards & rows

Goal row, run row, list row

Most lists in Aleph are rows of this shape: a small interactive element on the left (checkbox, status stripe), the title and metadata in the center, a timestamp on the right. Hover lifts the border, never the elevation.
