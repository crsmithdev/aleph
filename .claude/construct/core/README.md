# construct-core

Foundation pack. Provides CLAUDE.md, settings.json, statusline, and optional identity files.

**Depends on:** nothing (always required)

## Contents

- `hooks/statusline.ts` — status bar: model, branch, dir, context %, tokens
- `identity/` — optional semantic identity layer:
  - `SOUL.md` — purpose, values, mental models
  - `IDENTITY.md` — name, tone, personality
  - `STYLE.md` — output formatting, conventions
  - `USER.md` — principal profile, environment
  - `BOOTSTRAP.md` — session initialization sequence

Post-install verification: see [INSTALL.md](INSTALL.md).
