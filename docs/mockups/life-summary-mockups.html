Life · Summary — concept mockups

Three approaches for showing today / yesterday / last week in a single data card without resorting to disabled tags or sub-14px text. Drop any of them in isolation, or mix (e.g. variant A for counts and variant C for momentum).

Variant A — Stacked rows, today is hero

The number scale itself does the de-emphasis: today is 32px, yesterday 18px, last week 16px. No opacity, no “disabled” token. Color tints today’s number per metric.

Variant B — Segmented bars (proportional)

All three periods sit in the same card with a horizontal track each. Bar opacity steps down (today full color, yesterday 70% mix, last week 40% mix) without using CSS opacity on text. The number column stays at 14px so it’s always readable.
