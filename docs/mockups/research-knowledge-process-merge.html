Graph tab — merged knowledge + process (three-pane, ultrawide)

Threads (process) and concepts (knowledge) are different views of the same underlying graph: threads spawn findings, findings compose concepts, concepts link back to threads. This tab merges them into a single three-pane workspace tuned for ultrawide displays — tree on the left, canvas in the middle, inspector on the right. The default state is fully populated on first paint: tree expanded one level from every seed, canvas pre-rendered with the seed thread focused, inspector showing the seed thread's findings.

★ Implementation gate · default state must be useful at first paint

On a fresh open of the Graph tab, the user must see something useful without clicking, expanding, or resizing anything. Past iterations of the knowledge view have shipped with empty graphs that required user action before any data appeared — that's the regression this gate prevents.

  • Tree (left): auto-expand to depth 1 from every seed thread on mount. Concept children appear under their parent thread.
  • Canvas (center): pick the lens with data — Concepts · graph when concepts exist, otherwise Threads · map. Pre-render with the seed thread focused; never start with a blank or "click to load" state.
  • Inspector (right): when nothing is selected, show the seed thread's top findings (still useful, never blank).
  • Ultrawide ≥ 1600px: all three panes visible simultaneously, sized 320px / flex / 360px.
  • Below 1600px: inspector collapses to a right-side drawer that opens on selection.
  • Verification. bun run ui:smoke on a fixture session asserts: tree has visible depth-1 rows, canvas has rendered > 0 nodes, inspector shows non-empty content — all without simulating a click.

See docs/research-ui-rollout.md § 4 Stream 2D for the full spec.

Ultrawide layout. Page wrapper is widened to 2400px so the mockup reflects what the tab will look like on a real ultrawide. Three columns are 320px / flex / 360px. The center pane absorbs whatever's left — on a 2560px viewport that's ~1880px of canvas, plenty of room for the cytoscape graph to breathe. Below 1600px the inspector hides via media query (drawer-style on click).

Mockup follows the Material Palenight palette and the app's Noto Sans / Merriweather / Noto Sans Mono stack. Counts and dates are illustrative.
