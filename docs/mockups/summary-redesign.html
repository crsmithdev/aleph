Summary — redesign explorations

Three full-page directions for /summary. Same tokens, same data, different information architecture.

Variation A — Today hero + period tabs

Today is the headline. Goals get a prominent checked list, todos a compact checklist, habits a dot strip. Yesterday / Last 7 days collapse behind tabs so the page stops repeating itself.

Variation C — Type-first board

Pivot the grid: columns are Goals / Todos / Habits, rows are the periods. Each type owns its native shape — goals a list, todos a count table, habits a dot matrix. Git is purely a top datacard.
